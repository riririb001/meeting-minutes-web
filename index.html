📖 Groq API Key 발급 매뉴얼

이 서비스는 Groq API를 사용하여 음성을 텍스트로 변환하고 회의록을 자동 생성합니다.

API Key는 무료로 발급받을 수 있으며, 아래 단계를 따라하시면 됩니다.

⚠️ 참고: API 키는 이 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.
브라우저 데이터(캐시)를 삭제하면 키와 회의록이 함께 삭제됩니다.

자주 묻는 질문

비용이 발생하나요?

Groq API는 무료 티어를 제공합니다. 일반적인 회의록 사용량에서는 비용이 발생하지 않습니다.

API 키를 분실했어요

Groq 콘솔에서 기존 키를 삭제하고 새로 발급받으면 됩니다. 위 Step 2~4를 다시 진행하세요.

다른 기기에서도 사용할 수 있나요?

같은 API 키를 다른 기기 브라우저에 입력하면 됩니다. 단, 회의록 데이터는 기기별로 별도 저장됩니다.
