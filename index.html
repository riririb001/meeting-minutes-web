Groq API 키 발급 방법

Groq는 무료로 사용할 수 있는 AI API 서비스입니다.

  1. Groq Console 접속
    https://console.groq.com 에 접속합니다.
  2. 회원가입 / 로그인
    Google 계정으로 간편하게 가입할 수 있습니다.
  3. API Keys 메뉴 이동
    좌측 메뉴에서 API Keys를 클릭합니다.
  4. 새 키 생성
    Create API Key 버튼을 누르고, 이름을 입력한 뒤 생성합니다.
  5. 키 복사 & 붙여넣기
    생성된 키(gsk_로 시작)를 복사하여 왼쪽 사이드바의 API Key 입력란에 붙여넣으세요.

참고: Groq 무료 플랜으로 충분히 사용 가능합니다. API 키는 이 브라우저에만 저장되며 외부로 전송되지 않습니다.
