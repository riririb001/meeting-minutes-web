4

❓ 자주 묻는 질문

비용이 발생하나요?

Groq API는 무료 티어를 제공합니다. 일반적인 회의록 사용량에서는 비용이 발생하지 않습니다.

API 키를 잃어버렸어요

Groq 콘솔에서 기존 키를 삭제하고 새로 만들면 됩니다.

다른 기기에서도 사용할 수 있나요?

같은 API 키를 다른 기기 브라우저에 입력하면 됩니다. 단, 회의록 데이터는 기기별로 별도 저장됩니다.

브라우저 데이터를 삭제하면 어떻게 되나요?

API 키와 저장된 회의록이 함께 삭제됩니다. 키는 Groq 콘솔에서 다시 확인하거나 새로 발급받으세요.
